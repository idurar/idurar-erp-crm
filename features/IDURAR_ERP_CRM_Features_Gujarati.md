# 🧩 IDURAR Open-Source ERP & CRM

**IDURAR** is a powerful and free **Open-Source ERP & CRM Software** built on the **MERN Stack** (MongoDB, Express.js, React.js, Node.js).  
It provides complete business management tools, including user management, CRM, sales, inventory, purchase, financial, and project management modules.

## ⚙️ Software Stack

| Component | Description |
|------------|-------------|
| **Node.js** | JavaScript runtime for backend |
| **Express.js** | Web framework for Node.js |
| **React.js** | Frontend library for building UI |
| **Redux** | State management for React |
| **MongoDB** | NoSQL database |
| **Ant Design (AntD)** | UI component framework |

## ✨ Key Features

### 👥 User Management
- Create, edit, and delete user accounts  
- Role-Based Access Control (RBAC)  
- Secure authentication and authorization

### 💼 CRM (Customer Relationship Management)
- Manage leads, prospects, and customers  
- Track communication history (emails, calls, meetings)  
- Schedule appointments and send reminders  

### 💰 Sales Management
- Manage sales orders and invoices  
- Inventory tracking and stock updates  
- Integration with payment gateways  
- Sales performance dashboards and reports  

### 🛒 Purchase Management
- Create and manage purchase orders  
- Track supplier information  
- Update inventory upon receiving goods  
- Manage supplier invoices and payments  

### 📦 Inventory Management
- Stock tracking and adjustments  
- Low-stock alerts and auto purchase order creation  
- Barcode scanning support  
- Product categorization and pricing  

### 💳 Financial Management
- Financial ledger system  
- Accounts Receivable & Payable tracking  
- Balance sheets and income statements  
- Integration with accounting software  

### 📁 Project Management
- Create and track projects  
- Assign tasks and deadlines  
- Monitor progress and project costs  
- Real-time collaboration and document sharing  

### 📊 Reporting & Analytics
- Generate detailed business reports  
- Customizable dashboards for KPIs  
- Data visualization tools  

### 🔌 Integrations & Customization
- Third-party app integrations (Email tools, APIs)  
- Webhooks for data synchronization  
- Customizable UI and modules  

### 💻 User-Friendly Interface
- Built with React.js + Ant Design  
- Intuitive navigation and responsive UI  
- Supports multiple screen sizes
