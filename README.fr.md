# Ce projet a été créé avec [Create React App](https://github.com/facebook/create-react-app).

## Scripts disponibles

Dans le répertoire du projet, vous pouvez exécuter :

### `yarn start`

Lance l’application en mode développement.<br />
Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

La page se rechargera automatiquement lors des modifications.<br />
Les erreurs de lint apparaîtront dans la console.

### `yarn test`

Lance le test runner en mode interactif.<br />
Voir la section [exécution des tests](https://facebook.github.io/create-react-app/docs/running-tests).

### `yarn build`

Construit l’application pour la production dans le dossier `build`.<br />
Optimisée et prête à être déployée.

### `yarn eject`

**Attention :** cette opération est irréversible. Une fois `eject` exécuté, vous ne pouvez plus revenir en arrière.
