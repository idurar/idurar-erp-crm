# இந்த திட்டம் [Create React App](https://github.com/facebook/create-react-app) மூலம் உருவாக்கப்பட்டது

## கிடைக்கும் கட்டளைகள்

திட்ட அடைவில், நீங்கள் இயக்கலாம்:

### `yarn start`

பயன்பாட்டை மேம்பாட்டு முறையில் இயக்குகிறது.<br />
[http://localhost:3000](http://localhost:3000) என்ற முகவரியில் உலாவியில் காணலாம்.

நீங்கள் திருத்தங்கள் செய்தால் பக்கம் தானாக மீளேறும்.<br />
கான்சோலில் ஏதேனும் லிண்ட் பிழைகள் இருந்தால் காணலாம்.

### `yarn test`

பயன்பாட்டு சோதனையாளர் (test runner) இடைமுக பார்வை முறையில் இயக்குகிறது.<br />
மேலும் விவரங்களுக்கு [running tests](https://facebook.github.io/create-react-app/docs/running-tests) பகுதியை பார்க்கவும்.

### `yarn build`

பயன்பாட்டை உற்பத்தி (production) பயன்பாட்டிற்காக `build` கோப்பகத்தில் தொகுக்கிறது.<br />
React பயன்பாட்டை சிறப்பாக தொகுத்து செயல்திறன் மேம்படுத்தப்படுகிறது.

தொகுப்பு சுருக்கப்பட்டு, கோப்பு பெயர்கள் ஹாஷ் கொண்டிருக்கும்.<br />
உங்கள் பயன்பாடு வெளியீட்டிற்கு தயாராக உள்ளது!

மேலும் விவரங்களுக்கு [deployment](https://facebook.github.io/create-react-app/docs/deployment) பகுதியை பார்க்கவும்.

### `yarn eject`

**குறிப்பு:** இது ஒரே வழி செயல் — ஒருமுறை `eject` செய்தால் மீண்டும் திரும்ப முடியாது!
