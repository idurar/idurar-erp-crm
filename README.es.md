# Este proyecto fue creado con [Create React App](https://github.com/facebook/create-react-app).

## Scripts disponibles

En el directorio del proyecto, puedes ejecutar:

### `yarn start`

Ejecuta la aplicación en modo desarrollo.<br />
Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

La página se recargará automáticamente cuando realices cambios.<br />
También verás errores de lint en la consola.

### `yarn test`

Inicia el test runner en modo interactivo.<br />
Consulta [ejecutar pruebas](https://facebook.github.io/create-react-app/docs/running-tests).

### `yarn build`

Crea la aplicación para producción en la carpeta `build`.<br />
Optimiza la compilación para un mejor rendimiento.

### `yarn eject`

**Nota:** esta es una operación irreversible. Una vez que ejecutes `eject`, no podrás deshacerlo.
