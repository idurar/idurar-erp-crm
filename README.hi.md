# यह प्रोजेक्ट [Create React App](https://github.com/facebook/create-react-app) से बनाया गया है।

## उपलब्ध स्क्रिप्ट्स

प्रोजेक्ट डायरेक्टरी में आप चला सकते हैं:

### `yarn start`

डेवलपमेंट मोड में ऐप चलाता है।<br />
ब्राउज़र में देखने के लिए [http://localhost:3000](http://localhost:3000) खोलें।

पेज एडिट करने पर अपने आप रीलोड हो जाएगा।<br />
कंसोल में किसी भी लिंट एरर को देख सकते हैं।

### `yarn test`

इंटरएक्टिव वॉच मोड में टेस्ट रनर शुरू करता है।<br />
अधिक जानकारी के लिए [running tests](https://facebook.github.io/create-react-app/docs/running-tests) देखें।

### `yarn build`

प्रोडक्शन के लिए ऐप को `build` फोल्डर में तैयार करता है।<br />
यह React को प्रोडक्शन मोड में बंडल करता है और प्रदर्शन को अनुकूलित करता है।

### `yarn eject`

**ध्यान दें:** यह एक-तरफ़ा प्रक्रिया है — एक बार `eject` करने के बाद वापस नहीं जा सकते।
